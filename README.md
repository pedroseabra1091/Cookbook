# Cookbook ![crappy wooden spoon](crappy-wooden-spoon.png)

This is a playground created to explore Ecto's capabilities during Elixir session # 5. 

Addresses the following topics:
- Migrations
- Schemas
- Repo
- Changesets
- cast_assoc vs put_assoc

Future work:
- Multi
- Transaction
